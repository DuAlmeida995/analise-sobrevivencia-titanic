# analise-sobrevivencia-titanic
Projeto de análise exploratória de dados (EDA) sobre o dataset do Titanic. O objetivo é investigar a hipótese de que fatores socioeconômicos (classe, tarifa) e demográficos (sexo, idade) influenciaram a probabilidade de sobrevivência dos passageiros. Feito com Pandas, Seaborn e Matplotlib.
